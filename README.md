Bistro De Eiken — Restaurant Website

Live site: bistro-de-eiken.netlify.app

A warm, elegant one-page restaurant website for a fictional bistro in Enschede. Built as a portfolio project to demonstrate web development skills with a focus on visual design and interactivity.

📋 Project Overview
TypeFictief portfolio projectNicheRestaurant & caféStackHTML5, CSS3, Vanilla JavaScriptDesignOne-pager, warm & elegant, mobile-first

🛠️ Features

Sticky navbar met hamburger menu op mobiel
Full-screen hero met animated scroll indicator
Kenmerken sectie (4 USPs)
Interactief menu met tabs (Voorgerechten, Hoofdgerechten, Desserts, Dranken)
Over ons sectie met overlappende afbeeldingen en statistieken
Sfeer gallerij grid
Reserveringsformulier met datumvalidatie en success state
Openingstijden overzicht
Scroll animaties via Intersection Observer
Volledig responsive (mobiel, tablet, desktop)


🧰 Tech Stack

HTML5 (semantische structuur)
CSS3 (custom properties, grid, flexbox, animaties)
Vanilla JavaScript (tab systeem, form handling, scroll animaties)
Google Fonts (Playfair Display + Lato)
